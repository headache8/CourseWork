## 📝 Описание
Игрок размещает случайные фигуры на поле 8×8. При заполнении строки или столбца линия исчезает с плавной анимацией, начисляются очки и комбо-бонус. В режиме «На время» очистка линий даёт дополнительные секунды. Игра сохраняет прогресс и таблицу рекордов в локальный файл.

## 🛠 Функционал
| Основной | Дополнительный |
|----------|----------------|
| Поле 8×8 с 4 цветовыми темами | Таблица рекордов (топ-5, связный список) |
| Drag & Drop мышью | Сохранение/загрузка состояния (`save.dat`) |
| Очистка линий с анимацией затухания | Комбо-система + бонусное время |
| 2 режима: Классика / На время (45с) | 3 фигуры на панели, генерация по шаблону |
| Экраны: Титульный, Меню, Игра, Game Over | Подсказки управления в интерфейсе |

## 🎮 Управление
| Клавиша / Действие | Функция |
|--------------------|---------|
| `ЛКМ + перетаскивание` | Размещение фигуры на поле |
| `Пробел` | Переход с титульного экрана в меню |
| `ESC` | Выход в меню / закрытие программы |
| `R` | Рестарт после Game Over |
| `S` | Сохранить игру |
| `T` | Сменить тему поля |

## 📦 Сборка и запуск
Visual Studio 2026
1. Установи [SDL2](https://www.libsdl.org/download-2.0.php) и [SDL_ttf](https://www.libsdl.org/projects/SDL_ttf/).
2. Создай консольный C++ проект → добавь `main.cpp`, `game.cpp`, `game.h`.
3. В `Проект → Свойства → VC++ Directories`:
   - `Include Directories`: путь к `SDL2/include`, `SDL_ttf/include`
   - `Library Directories`: путь к `SDL2/lib/x64`, `SDL_ttf/lib/x64`
4. `Linker → Input → Additional Dependencies`: `SDL2.lib`, `SDL2main.lib`, `SDL2_ttf.lib`
5. Скопируй `SDL2.dll` и `SDL2_ttf.dll` в папку с `.exe`. Положи `arial.ttf` в корень.

Для быстрого запуска без настройки среды скачай установщик:

Установщик включает:
- ✅ Скомпилированную игру `.exe`
- ✅ Все зависимости (`SDL2.dll`, `SDL2_ttf.dll`)
- ✅ Шрифт `arial.ttf`
- ✅ Ярлык на рабочем столе (опционально)
- ✅ Автоматическую регистрацию расширений (если нужно)

> 💡 Файл `save.dat` и таблица рекордов создаются при первом запуске в папке `%APPDATA%\GameBlocks\` (или рядом с `.exe`).
